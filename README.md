# 🎓 Lupanulla Elimu Hub

**Kitovu cha Elimu Tanzania**

Lupanulla Elimu Hub is a modern AI-powered educational platform designed to provide students, teachers, authors, and educational institutions with access to learning resources, online courses, examinations, AI tutoring, digital materials, and educational opportunities.

## 🌍 Vision

To become Tanzania's leading digital education ecosystem by connecting learners, educators, and educational resources through technology and artificial intelligence.

## 🚀 Getting Started

### Prerequisites

* Node.js 18+
* npm or yarn
* Google Gemini API Key
* Supabase Project

### Installation

Clone the repository:

```bash
git clone https://github.com/your-username/lupanulla-elimu-hub.git
cd lupanulla-elimu-hub
```

Install dependencies:

```bash
npm install
```

Create a `.env.local` file:

```env
GEMINI_API_KEY=your_gemini_api_key

VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key

VITE_ADSENSE_CLIENT_ID=your_adsense_client_id
VITE_ADSENSE_SLOT_ID=your_adsense_slot_id
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```
